# ELK_Stack
ELK Stack auto install

Cilj projekta:

Napraviti automatsku instalaciju ELK (Elasticsearch-Logstash-Kibana) Stack-a
Sastojat ce se od dva dijela: klijentskog i serverskog

Klijent:
- topbeat
- filebeat
- packetbeat
- NGINX

Server:
- Java
- Elasticsearch
- Logstash
- Kibana
