# ELK_Stack
ELK Stack auto install

Automation of ELK Stack

Cilj projekta:

Napraviti automatsku instalaciju ELK (Elasticsearch-Logstash-Kibana) Stack-a

Sastojat ce se od dva djela: klijentskog i serverskog

Klijent:
- topbeat
- filebeat
- packetbeat
- NGINX

Server:
- Java
- Elasticsearch
- Logstash
- Kibana
